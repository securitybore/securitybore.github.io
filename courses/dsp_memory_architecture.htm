DSP memory architecture class

1-day industrial short course

This 1-day course explains how the memory and cache architecture of a system affects its use for DSP, and how to transform code so that the memory is used most efficiently.

You will learn to understand:

  • How to estimate memory access needs for an application
  • Typical DSP memory architectures
  • Typical DSP cache architectures
  • How to transform code to use memory efficiently

Class content:

  • Signals and Noise
  • Processing Gain
  • Designing DSP specifications
  • Frequency domain
  • Window functions
  • Sampling
  • Sizing
  • Filtering
  • Filtering to enhance signal to noise ratio
  • Filtering applications
  • FIR filter design
