TriMedia overview - index

TriMedia enables multimedia in consumer devices. TriMedia-based chips integrate a high-speed Media Processing CPU core with 'smart' peripherals for audio, video, graphics, control and communications in a single inexpensive System on Chip. TriMedia is programmable, so that it can be adapted to many different applications and so that products can be changed through software to meet evolving standards and requirements. An object-oriented TriMedia Software Architecture makes it possible to design new applications by linking of high-level software components, and promotes a high degree of software re-use.

TriMedia is the base for Media Processing chips including: Philips pnx1300 and pnx1500 ('Nexperia)

This brief overview aims to introduce and explain the design considerations behind TriMedia, and show how the needs of OEM developers in rapidly changing multimedia markets are addressed.
