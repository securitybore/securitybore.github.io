TriMedia overview - multimedia for multiple media

TriMedia targets diverse markets for multimedia products. It is optimized to work efficiently as the base for varied products. Examples of succesful products based on TriMedia include DVD, digital TV, standard TV enhancements, video security systems, home networking, video mail, internet radio, and video editing systems.

Many more new products are possible because of the TriMedia's great flexibility and programmability.

TriMedia is called a 'processor' because it deals efficiently with multiple media. It has four key design features:

  1. a programmable core optimized for multimedia
  2. support for 'control' and data transmission as well as DSP
  3. peripherals for audio, video, graphics, control & communications
  4. Software Architecture that provides multimedia functions

Consumer multimedia is in a state of change: new products, and new standards, emerging all the time. TriMedia unites the common key features that are need, and can be programmed to match changing standards and to suit different markets or applications.

The TriMedia Software Architecture also helps in this, by making it easier to re-use software modules in new products.
