TriMedia overview - technical overview

Chips for consumer multimedia need to be cheap, but they also involve high speed and specialized processing and I/O. TriMedia's 'System on Chip' architecture integrates all of the computation, and the multimedia peripherals, onto a single chip so that the overall system cost can be greatly reduced.

There are three major elements to the TriMedia System on Chip architecture:

  • TriMedia CPU core - optimized for media processing
  • co-processors - greatly accelerating video processing
  • I/O peripherals - efficient I/O integrated 'on chip'

The System on Chip architecture is also anchored by the TriMedia Software Architecture - an object-oriented software approach that makes it possible to design systems at an abstract software functional level.

Integrated co-processors and 'smart' peripherals are central to the TriMedia architecture - both allow some very demanding parts of targetted applications to be greatly accelerated through specialized and dedicated hardware, while leaving the fully programmable CPU core to implement more versatile functions.
